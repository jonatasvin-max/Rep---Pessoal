# Projeto CSS Zen Garden

## Autor
Jônatas Cardoso

## Tema
Preto e Branco Minimalista

## Design de Referência
Verde Moderna (Design #214)

## Link de Referência
https://www.csszengarden.com/214/

## Descrição
Este projeto recria o layout do CSS Zen Garden utilizando exclusivamente Flexbox, inspirado no design #214, mas adaptado para uma paleta monocromática em preto e branco. 

## Recursos Utilizados
- Flexbox
- Variáveis CSS (:root)
- Media Queries
- Reset CSS
- Tipografia personalizada
- Paleta de cores consistente (preto e branco)
- Efeitos :hover
- Animações com @keyframes e transitions
- Componentes reutilizáveis
- Modo claro/escuro
- Comentários organizando o código
