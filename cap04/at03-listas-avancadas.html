  1. HTML e semântica
    1. Estrutura básica
    2. Elementos semânticos
  2. Tabelas
    1. Estrutura básica

Nesta primeira etapa do roteiro de estudos, o foco está nos fundamentos essenciais do desenvolvimento web. São abordados os conceitos de HTML e semântica, a construção de tabelas e o uso de listas e recursos multimídia.

2. Ranking de Tecnologias

  1. HTML (HyperText Markup Language)

    É a linguagem de marcação padrão para estruturar páginas web.

  2. CSS (Cascading Style Sheets)

    Responsável pela estilização e design das páginas.

  3. JavaScript

    Linguagem de programação que adiciona interatividade às páginas web.

  4. Banco de Dados (SQL e NoSQL)

    Essenciais para armazenar e gerenciar informações em aplicações web.

  5. Frameworks e Bibliotecas (React, Angular, Vue)

    Facilitam o desenvolvimento de interfaces dinâmicas e complexas.

3. Glossário Técnico

JavaScript
Linguagem de programação que adiciona interatividade.
DNS
Servidor de nomes.
Sistema que traduz nomes de domínio em endereços IP
URL
Endereço web
Identificador único de um recurso na internet.
HTTP
Protocolo usado para comunicação na web.
Também pode indicar a versão do protocolo (HTTP/1.1, HTTP/2, HTTP/3).
API
Interface que permite a comunicação entre diferentes sistemas.
Conjunto de funções e rotinas disponibilizadas por uma aplicação.
CSS
Folhas de estilo que definem aparência e layout.
